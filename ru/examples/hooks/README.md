# Хуки принуждения (opt-in)

Правила и скиллы уговаривают агента; хуки делают ключевые нарушения технически невозможными. Правило живёт в контексте и теряет вес по мере роста сессии. Хук - это ваш код на событии жизненного цикла: он выполняется независимо от того, что модель сейчас «думает».

Три готовых гейта для Playwright-проектов:

| Хук | Событие | Что делает |
|---|---|---|
| `gate-commit.sh` | PreToolUse (Bash) | `git commit` не проходит без свежего зелёного `test-results/.last-run.json` |
| `track-test-edit.sh` | PostToolUse (Edit\|Write) | правка `*.spec.ts` поднимает флаг «тесты грязные» и вшивает агенту напоминание о прогоне |
| `gate-stop.sh` | Stop | агент не может завершить ход с грязными тестами; самоочистка после прогона; предохранитель не даёт блокировать больше двух раз подряд |

## Установка (копированием, не плагином)

Хуки **сознательно не входят в плагин** - гейт на коммит, молча включённый чужим пакетом, был бы враждебным. Подключение только руками:

```bash
mkdir -p .claude/hooks
cp ru/examples/hooks/*.sh .claude/hooks/ && chmod +x .claude/hooks/*.sh
# слить ru/examples/hooks/settings.json со своим .claude/settings.json
```

Требования: `jq`, Playwright (он сам пишет `test-results/.last-run.json` после каждого прогона).

## Проектные решения

- **Принцип асимметрии.** Гейт на коммит в сомнении запрещает (ложный блок стоит один `npx playwright test`; ложный пропуск - непрогнанный коммит в истории). Гейт на завершение в сомнении пропускает (ложный блок - риск вечного цикла).
- **Причина блока - это инструкция.** Агент читает её и действует по ней: пишите туда команду, которую нужно выполнить, а не «нельзя».
- **Предохранитель обязателен.** Агрессивный Stop-хук без лимита блокировок вешает сессию в цикле «стоп - блок», когда окружение сломано.
- **Хук проверяет форму, не смысл.** Зелёный прогон не доказывает осмысленность тестов - `expect(true).toBe(true)` пройдёт любой гейт. Смысл ловит ревью: скилл [`test-review`](../../skills/test-review).

Маски файлов (`*.spec.ts`, `tests/`) и путь к артефакту адаптируйте под свой проект.
